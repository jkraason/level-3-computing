Calculate U_LJ again for the new system

IF U_LJ2<U_LJ1: 
    reject move

IF U_LJ2>U_LJ1:
    accept move if rand < exp[(U_LJ1-U_LJ2)/k_bT]
    U_LJ1 = U_LJ2
    calculate energy again - reuse for next mc move

RUN until equilibrated
